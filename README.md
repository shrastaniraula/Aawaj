# Requirements

1. node version v16 or higher
2. npm 8.0 or Higher

## Getting Started

**Step 1.** Install all dependencies using <br/>
`npm install ` <br/>
**Step 2.** Start Development server using <br/>
`npm start` <br/>

**Step 3.** Install [Expo Go App](https://play.google.com/store/apps/details?id=host.exp.exponent&hl)in your phone and scan the given QR code.
You Should have working instance of the app running in your phone.
